# توثيق نظام مكتبة الوسائط (Media Library System)

مكتبة الوسائط هي وحدة برمجية متكاملة لإدارة ورفع وتعديل ونقل وحذف المجلدات والملفات المرفوعة على السيرفر بأسلوب تفاعلي متطور يشبه إلى حد كبير مكتبة وسائط WordPress، مع الالتزام الكامل بمعايير الأمان وتدويل النصوص (Localization) للعربية والإنجليزية.

---

## 2. العمليات والوظائف الأساسية (Core Operations)

| العملية | آلية العمل | التفاعل والأداء | المعالجة الخلفية |
| :--- | :--- | :--- | :--- |
| **الرفع (Upload)** | سحب وإفلات الملفات أو اختيارها يدوياً. | شريط تقدم AJAX تفاعلي يوضح النسبة المئوية لحظياً. | التحقق من الحجم الأقصى، فلترة الامتدادات الآمنة، وتوليد اسم فريد مضاف إليه تسلسل رقمي في حال وجود ملف بنفس الاسم. |
| **إنشاء مجلد جديد** | زر "مجلد جديد" يفتح نافذة منبثقة (Modal). | إدخال اسم مجلد مناسب والتحقق من صحته. | التحقق من عدم تكرار المجلد في المسار الحالي، وتطهير الاسم من الأحداث البرمجية الضارة والرموز الخاصة. |
| **تغيير الاسم (Rename)** | تحديد عنصر ثم النقر على زر "تغيير الاسم" في شريط الإدارة. | مودال إدخال نصي من SweetAlert2 مع إمكانية تعديل الاسم بدون الامتداد بالنسبة للملفات. | التحقق من صحة الاسم الجديد وعدم وجود ملف/مجلد آخر بنفس الاسم، ومن ثم تعديل الاسم مادياً على القرص. |
| **النقل (Move)** | تحديد عنصر ثم النقر على زر "نقل" في شريط الإدارة. | مودال اختيار منسدل (Select) من SweetAlert2 يعرض شجرة المجلدات المتوفرة فقط. | التحقق من منع النقل الدائري (عدم إمكانية نقل مجلد إلى داخل نفسه أو فروع تابعة له)، ثم نقل الملف/المجلد بالكامل. |
| **الحذف (Delete)** | تحديد عنصر والنقر على "حذف نهائي" من الجانب، أو زر السلة. | نافذة تأكيد تحذيرية من SweetAlert2 لمنع الحذف بالخطأ. | حذف الملف بشكل نهائي أو المجلد بكافة محتوياته وفروعه من القرص باستخدام `Directory.Delete(path, true)`. |
| **التحميل (Download)** | النقر على زر التحميل في شريط الأدوات أو عارض التفاصيل. | تنزيل مباشر للملف عبر المتصفح. | إحالة رابط الملف مباشرة للمتصفح مع السمة `download`. |

## 3. قواعد الأمان وصلاحية المجلدات السنوية (Security & Year Folder Rules)

| القاعدة / الشرط | الواجهة الأمامية (Frontend Validation) | المعالجة الخلفية (Backend Validation) | الأثر والهدف |
| :--- | :--- | :--- | :--- |
| **مجلدات الأرشيف السنوية (2026 - 2035)** | يُمنع يدوياً إنشاؤها أو حذفها أو تعديل اسمها أو نقلها في مستوى الجذر الرئيسي لـ uploads. | يُرفض تماماً أي طلب يدوي لحذفها أو تعديل اسمها أو نقلها عبر فحص المسار الرئيسي بالـ C#. | حماية الهيكل التنظيمي للملفات السنوية المعتمدة تلقائياً من التخريب أو العبث اليدوي. |
| **الإنشاء التلقائي للأرشيف** | غير مطبق (يتم بشكل خلفي تماماً). | عند قيام المحرر برفع صور أو وسائط للمقالات، ينظر النظام للسنة الحالية وينشئ المجلد آلياً إن لم يكن متوفراً. | محاكاة أسلوب وردبريس في أرشفة الملفات بالسنوات تلقائياً وبأمان تام دون تدخل المستخدم. |
| **فحص الامتدادات الممنوعة** | فلترة الحقول في منتقي الملفات. | حظر صارم للملفات التنفيذية والبرمجية (`exe`, `dll`, `bat`, `cmd`, `sh`, `php`, `asp`, `aspx`, `cshtml`, `js`, `config`, `json`). | حماية السيرفر من هجمات رفع الملفات الخبيثة (Remote Code Execution) التي قد تؤدي لاختراق النظام. |
| **تحديد الحجم الأقصى للرفع** | التحقق الأولي من حجم الملفات المحددة. | قراءة حجم الملف والتحقق من أنه لا يتجاوز الحد المسموح به في جدول `Settings` في قاعدة البيانات (مثل 20 ميجابايت). | منع إغراق مساحة تخزين السيرفر بالملفات الضخمة أو التسبب في نفاد الذاكرة العشوائية. |
| **تأكيد الهوية والصلاحيات** | إخفاء الأزرار المتقدمة في حال عدم توفر الصلاحية. | فحصClaims الصلاحيات (`Media.View`, `Media.Upload`, `Media.Delete`) قبل كل عملية. | ضمان أن المستخدمين المصرح لهم فقط هم من يمكنهم تعديل أو إدارة الوسائط داخل النظام. |

## 4. قاموس ومفاتيح الترجمة المخصصة للميديا (Localization Keys Dictionary)

مكتبة الوسائط مجهزة لدعم ثنائي اللغة بالكامل بالاعتماد على حدث `tp-lang-changed` المشترك، ويتم ترجمة الواجهة فورياً دون الحاجة لتحديث الصفحة بالاعتماد على المفاتيح التالية:

| مفتاح الترجمة (Key) | القيمة بالعربية (AR) | القيمة بالإنجليزية (EN) | موضع الاستخدام في الواجهة |
| :--- | :--- | :--- | :--- |
| `mediaTitle` | مكتبة الوسائط | Media Library | العنوان الرئيسي لترويسة الصفحة. |
| `mediaNewFolder` | مجلد جديد | New Folder | زر إنشاء مجلد جديد العلوي. |
| `mediaUploadFiles` | رفع ملفات | Upload Files | زر رفع الملفات العلوي. |
| `mediaSearchPlaceholder` | بحث في المجلد الحالي والفرعي... | Search current folder & subfolders... | نص التلميح داخل شريط البحث. |
| `mediaEmptyStateTitle` | لا توجد ملفات أو مجلدات في هذا المسار. | No files or folders found in this directory. | ترويسة الحالة الفارغة للمجلد. |
| `mediaEmptyStateDesc` | اسحب الملفات وأفلتها هنا للرفع الفوري أو اضغط على رفع ملفات. | Drag & drop files here for instant upload or click Upload Files. | وصف الحالة الفارغة للمجلد. |
| `mediaDetailsTitle` | تفاصيل الملف | File Details | عنوان اللوحة الجانبية عند تحديد ملف. |
| `mediaFolderDetailsTitle` | تفاصيل المجلد | Folder Details | عنوان اللوحة الجانبية عند تحديد مجلد. |

| `mediaSelectToPreview` | حدد ملفاً أو مجلداً لمعاينة التفاصيل هنا. | Select a file or folder to preview details here. | رسالة اللوحة الجانبية عند عدم اختيار أي عنصر. |
| `mediaToolbarOptions` | خيارات الإدارة: | Management Options: | النص التعريفي لشريط الأدوات السريع. |
| `mediaToolbarRename` | تغيير الاسم | Rename | زر خيار إعادة التسمية في شريط الأدوات. |
